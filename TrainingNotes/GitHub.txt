GitHub
------
- Git areas : 

	1) Working Directory 

	2) Staging area 		
		
		to add the files to the staging area : git add file_name

		but to add all files and changes in files at a time : git add .

	3) Local Repository : the clone of your original file

		to add it to the local repository : git commit -m "message"

	4) Remote Repository 

		to add to the remote repository : git push origin main



git add file_name
|
git add .
|
git commit -m "message"
|
git push origin main


- git checkout file_name : used to undo the changes which you added in working directory



Create a branch and move to branch
----------------------------------

- git branch branch_name : to create branch

- git checkout branch_name : to switch the branch
